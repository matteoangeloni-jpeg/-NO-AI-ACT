How to cite NO AI ACT

If you use NO AI ACT in a thesis, a paper, a classroom study or derivative work, this page gives you the correct citation formats. Always state the exact version you used: the game evolves, and case content can change between versions.

01APA citation

APA (7th edition) format for software:

Angeloni, M. (2026). NO AI ACT — Simulatore di una società non regolata (Version 2.0.0) [Software]. https://www.no-ai-act.eu/

Replace "2.0.0" with the version you actually used: it is shown in the site footer, on the game's title screen, and in the repository's package.json.

02BibTeX

@software{angeloni_no_ai_act_2026,
  author  = {Angeloni, Matteo},
  title   = {NO AI ACT --- Simulatore di una societ{\`a} non regolata},
  year    = {2026},
  version = {2.0.0},
  url     = {https://www.no-ai-act.eu/},
  note    = {Source code: https://github.com/matteoangeloni-jpeg/-NO-AI-ACT.
             MIT-licensed code; educational content CC BY 4.0.}
}

03Why the exact version matters

  • Reproducibility: cases, texts and scoring rules are pinned per version. Anyone replicating a study must be able to retrieve exactly what participants saw.
  • Where to find it: the site footer, package.json, and the release tags on GitHub Releases.
  • Historical archive: every tagged GitHub release preserves the code and content of that version.

04DOI

A DOI (Digital Object Identifier) is not yet available. The project is planned to be archived on Zenodo with the 2.0 release: once the DOI is live it will be published here, in the CITATION.cff file and in the repository README. Until then, cite the canonical URL and the version.

05The CITATION.cff file

The repository ships a CITATION.cff file in the Citation File Format: GitHub reads it automatically and shows the "Cite this repository" button, and reference managers can import it directly.

07Context for citing authors

NO AI ACT is an educational simulation of Regulation (EU) 2024/1689: it deliberately simplifies the law and is not legal advice. Its educational effectiveness has not yet been empirically validated: in publications, describe it as an open-source educational tool, not as an intervention with demonstrated effectiveness. The full methodological framework is on research and methodology; media materials are in the press kit.
